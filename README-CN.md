## 安装使用

- 获取项目代码

```bash
git clone https://github.com.cnpmjs.org/dgiot/dgiot-dashboard.git
```

- 安装依赖

```bash
cd dgiot-dashboard

npm i -g pnpm --registry=https://registry.npmmirror.com

pnpm config set registry https://registry.npmmirror.com

pnpm -v

pnpm install
```

- 运行

```bash
pnpm dev
```
